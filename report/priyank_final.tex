\section{Introduction}
We have been provided with the MNIST dataset, which consists of 42,000 training images. Each image is a 28 by 28 pixel grayscale image of a handwritten digit (0-9). The task is to classify these images into their respective digit classes. We will compare the performance of classical machine learning models using Logistic regression and with deep learning models (\textbf{Convolutional Neural Network}) on this dataset.
